1.	Are the HTML tags and elements are the same thing?

2.	What are tags and attributes in HTML?
Ans.	Tags are written in between the angular brackets. Some tags needs to close and some doesn't.
	HTML attributes are words which provide additional information about the elements. 

3.	What are the void element in HTML?
Ans.	The element which doesn't required closing tag is called void element.

4.	What are HTML Entities?
Ans.	An HTML entity is used to display invisible characters and reserved characters that would otherwise be interpreted as HTML code.
	Entities start with ampersand(&) and end with semicolon(;).

6.	What is the 'class' attribute in HTML?
Ans.	Attribute 'class' means when developer needs to change same style on multiple element or needs more than one class to single element, 'class' attribute is used.

7.	What is the different between the 'id' attribute and the 'class' attribute of HTML elements?
Ans.	id     : One element has only one unique id and one id cann't be repeated to other element.
	class : One element has more than one class and one class can be repeated to other element.

9.	How is Cell Padding different from Cell Spacing?
Ans.	Cell Padding : Cell padding is the space between the cell and the inner content. By default padding is 0 or according to browser.
	Cell Spacing : Cell spacing is the space between two cells. By default spacing is 2px.

16. 	How are active links different from normal links? 
Ans. 	Some link has some focus effect or click effect is called active link.

17.	What are the different tags to separate sections of text? 
Ans.	Three tags to separate section of text.
	1) <br>	      : By this tag, line will brake and jump to next line.
	2) <p>	      : By this tag, text will contain in new paragraph.

18.	What is SVG? 
Ans.	SVG is stands for Scalable Vector Graphics. It is invented by W3C in 1998. 
	It's an XML based markup language for two dimentional vector graphics. It can be scaled in or out without loosing any resolution.
	
19.	What is difference between HTML and XHTML? 
Ans.	HTML stands for Hyper Text Markup Language and XHTML stands for EXtensible Hyper Text Markup Language.
	In HTML, the attribute and tag names are case-insensitive while in XHTML attributes and tag must written in lower case.
	HTML is not strict to written while XHTML is highly strict to written.
	
20.	What are logical and physical tags in HTML?
Ans.	Logical tags  : Logical Tags are used in HTML to display the text according to the logical styles. 
			  Ex. <em>, <strong>
	Physical tags : Physical Tags are used in HTML to provide actual physical formatting to the text.
			  Ex. <i>, <b>
